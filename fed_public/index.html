Braid: Federation

This server is running braid-server

Braid servers connect clients running multi-user collaborative apps. These apps can run on virtually any device and interoperate with anyone else on this or any other reachable braid server.

Braid can be used on public or private networks. It is designed to federate, so that users in different domains using different servers can work together.

This port is used for federating between servers.
